# Installation

Follow these steps to install and set up the project.

## Prerequisites

- Node.js 18+ 
- npm or yarn
- Git

## Steps

1. Clone the repository
2. Install dependencies
3. Configure environment variables
4. Run the development server

## Next Steps

After installation, check out the [Configuration Guide](./configuration) to learn how to customize your setup.
