Authentication

This guide covers the implementation of authentication in the application.

## Overview

The application uses Firebase Authentication with support for:

- Email/Password authentication
- Google OAuth
- Role-based access control

## Implementation

Authentication is handled through the `lib/api/auth.js` module.
