# Introduction

Welcome to the documentation editor! This is a powerful tool for creating and managing your documentation.

## Features

- **Rich Text Editing**: Write with a powerful markdown editor
- **Organized Structure**: Organize your docs with categories and nested pages
- **Table of Contents**: Automatically generated from your headings
- **Easy Navigation**: Sidebar navigation with icons and categories

## Getting Started

1. Select a document from the sidebar
2. Edit the title and description
3. Write your content using the editor
4. Click **Save** to save your changes

The content is automatically saved as MDX files in the `content/modules` directory.
